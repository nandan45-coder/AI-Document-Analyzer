# AI Document Analyzer

AI-powered Document Analyzer built using FastAPI, ChromaDB, Sentence Transformers, and Google Gemini.

## Features

- PDF Upload
- Document Processing
- Text Chunking
- Embedding Generation
- ChromaDB Vector Storage
- Semantic Search
- RAG Chat with Documents
- Resume Analyzer

## Tech Stack

- FastAPI
- ChromaDB
- Sentence Transformers
- Gemini API
- SQLite
- Python
